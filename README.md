# NestJS GraphQL and PostgreSQL Demo 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Add  
`.env` file in the root folder with all your local variable  
  - Ex.
    ```
    ## PostgreSQL DATABASE
    DB_HOST=your_HOST
    DB_PORT=5432
    DB_NAME=PET_STORE
    DB_USER=your_DB_User
    DB_PASS=your_PASS
    ```
